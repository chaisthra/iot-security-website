About the Research

This research addresses the critical cybersecurity challenges faced by Small and Medium-sized Enterprises (SMEs) in protecting their Internet of Things (IoT) deployments.

Research Objectives

  • Identify and characterize the most relevant security threats to IoT deployments in SME environments
  • Determine critical network traffic metrics that effectively indicate different types of attacks
  • Develop machine learning models optimized for SME resource constraints
  • Create a tiered implementation framework based on available resources
  • Establish practical guidelines for model selection and alert handling

Dataset Information

The research utilizes the CIC-IoT-2023 dataset, which contains network traffic from various IoT devices under normal conditions and multiple attack scenarios. Key characteristics include:

  • Traffic from multiple IoT device types (cameras, thermostats, smart appliances, etc.)
  • Labeled attack scenarios including DDoS, reconnaissance, and spoofing attacks
  • Rich feature set with over 80 network traffic attributes
  • Both benign and malicious traffic patterns for comparative analysis
